# promo-app-Kevin-Mohamed

Ce répertoire permet l'affichage et l'ajout des students. 
Il est fait par Mohamed-Amine et Kevin.
Afin de démarrer le projet, il faut faire:
- composer update
- Il faut créer une base de données "student"
- et faire la commande "php vendor/bin/doctrine orm:schema-tool:create"
